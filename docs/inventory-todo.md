## スライス 1: 在庫の最低限 MVP

スライス終了時点で「出品者が在庫を設定でき、購入者は在庫がある商品オプションだけ買える」が動くようにする。

- [x] 在庫数を商品オプション単位で持つ
- [x] 出品者が在庫数を編集できる（API）
- [x] 出品者向け在庫管理画面（最低限）
- [x] 注文確定時に在庫 0 の商品オプションは注文不可（API ガード）
- [x] 在庫切れの商品オプションは購入者画面で注文ボタンを非活性にする（最低限）
- [x] 注文確定時に在庫を減らす

## スライス 2: 在庫切れ表示

- [x] 商品一覧で在庫切れ表示
- [x] カートで在庫切れ表示
- [x] 商品詳細で在庫切れ表示
- [x] 商品全体の在庫切れ判定（全商品オプションが在庫 0）

## スライス 3: カート追加時の在庫チェック

- [x] カート追加時に在庫チェックを行う
- [x] 在庫切れの商品オプションはカート追加ボタンを非活性にする

## スライス 4: キャンセル時の戻し

- [x] カート画面のレンダリング時に在庫を確認する
